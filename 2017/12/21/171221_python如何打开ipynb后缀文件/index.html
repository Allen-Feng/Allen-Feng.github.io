python的即时编辑是很多学习python人员的一个必备需求，毕竟谁也不想写完一整个文件，通过命令后运行，除了PyCharm这种IDE，使用ipynb也是一个不错的选择。

安装

需要安装ipython notebook，如果你已经安装Anaconda软件，则不需要安装。
如果未安装Anaconda，则需要安装，在控制台里输入：

打开ipynb文件

  1. cd 到 ipynb 文件所在目录
  2. 命令行输入ipython notebook, 浏览器中就会打开ipynb文件。

jupyter安装（可选）

网上搜索很多都要再安装jupyter
ipython notebook在下一个版本被命名为Jupyter Notebook。
通过pip install jupyter安装

但是在Mac环境实际操作时无需此步骤，在执行pip install “ipython[notebook]”时发现已经安装了。
